MWRB v3 - Textures Directory
=============================

Place your addon texture files in this directory.

Required texture files (based on shapes you want to use):

CIRCLE:
  - Charge_Empty.blp
  - Points_Fill_2.tga

SQUARE:
  - Square_Empty.blp
  - Square_Fill.tga

LIGHTNING 1:
  - Lightning1_Empty.blp
  - Lightning1_Fill.tga

LIGHTNING 2:
  - Lightning2_Empty.blp
  - Lightning2_Fill.tga

LIGHTNING 3:
  - Lightning3_Empty.blp
  - Lightning3_Fill.tga

LIGHTNING 4:
  - Lightning4_Empty.blp
  - Lightning4_Fill.tga

Note: Copy these from your existing MWRB installation's Textures folder.
The addon will fall back to circle textures if specific shapes are missing.
